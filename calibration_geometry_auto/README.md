# Automatic Geometry-Based Calibration

Метод выполняет **полностью автоматическое выравнивание камеры**, используя геометрию коробки.

## Что делает

- Находит линии (LSD detector)
- Делит их на вертикальные и горизонтальные
- Находит пересечения → углы коробки
- Строит перспективную матрицу
- Автоматически выравнивает изображение

Этот метод не требует:
- шахматной доски  
- ручных кликов  
- разметки  

## Использование

Поместите входной кадр в:

input.jpg


и запустите:

```bash
python calibrate_geometry_auto.py
```
Результат сохранится в output_auto.jpg.

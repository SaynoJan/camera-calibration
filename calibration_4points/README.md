# 4-Point Manual Camera Calibration

Этот метод позволяет быстро выровнять изображение, кликнув мышкой по четырём точкам.

## Что делает

- Пользователь выбирает 4 угла объекта или области
- Скрипт вычисляет перспективную матрицу (`homography`)
- Создаёт ровный вид сверху

## Как использовать

1. Поместите входное изображение в файл `input.jpg`
2. Запустите:

```bash
python calibrate_4points.py
```
3. Кликните последовательно 4 точки:

верхний левый

верхний правый

нижний правый

нижний левый

4. Итоговый файл: output_aligned.jpg

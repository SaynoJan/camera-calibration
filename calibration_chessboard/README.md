# Chessboard Camera Calibration

Этот метод выполняет **полную внутреннюю калибровку** камеры с помощью шахматной доски.

## Возможности

- Определяет матрицу камеры (`camera_matrix`)
- Вычисляет искажения линзы (`dist_coeffs`)
- Создаёт файл `camera.yaml`
- Подходит для точной геометрической обработки

## Как использовать

1. Распечатайте шахматную доску (PDF в папке `/resources`)
2. Положите её в область видимости камеры
3. Сделайте 10–20 снимков и положите их в папку `images/`
4. Запустите:

```bash
python calibrate_chessboard.py
```
5. Итоговый файл: camera.yaml
